void Set_Duty ( motor_t * p_mot,
int32_t speed )

Sets motor output based on signed speed.

Sets the motor speed and direction using a signed percentage.

Converts the speed percentage (âˆ’100 to +100) to a PWM duty cycle. The PWM resolution is defined by PWM_MAX_DUTY_CYCLE + 1. The scaling maps 100% to maximum PWM value.

Special case:

  • speed == 0: Sets both outputs to max duty, simulates full brake.
  • speed > 0: Drives forward using channel A.
  • speed < 0: Drives in reverse using channel B.
Parameters
p_motPointer to the motor object.
speedSigned percentage (-100 to 100). 0 = brake, >0 = forward, <0 = reverse.
