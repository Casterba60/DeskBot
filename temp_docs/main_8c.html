◆ main()

int main ( void )

Main program entry point.

This function initializes all hardware peripherals, sets up motors, PID controllers, and enters a loop that:

  • Performs joint homing using limit switches
  • Parses UART-based angle commands
  • Updates joint setpoints and servo angles
Return values
intAlways returns 0.
